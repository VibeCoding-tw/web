脈衝的藝術：
解密 PWM 占空比魔法

理解如何透過極速開關切換，讓數位腳位模擬出連續的類比電壓感。

🌉 1. 數位轉類比的橋樑：什麼是 PWM？

開關的藝術：電腦腳位本質上是數位的，只能輸出 0V (OFF) 或 3.3V (ON)。

PWM (Pulse Width Modulation) 是透過極快速地切換開關，讓外部設備感受到一個「平均電壓」。

占空比 (Duty Cycle) 的奧祕：

在一個固定週期內，「開啟時間」佔「總週期時間」的百分比。

  • 25% Duty Cycle 1/4 時間開，3/4 時間關 → 感覺像是 3.3V 的 25%。
  • 50% Duty Cycle 一半開，一半關 → 感覺像是 3.3V 的一半。

⚡ 2. 能量等效原理：平均電壓

核心數學公式：

\[V_{avg} = V_{max} \times \text{Duty Cycle}\]

視覺暫留與慣性： 為什麼我們看不到燈在閃？

因為切換速度（頻率）通常高達數千赫茲 (Hz)，人眼與馬達轉子因為慣性來不及反應，只會感受到平滑的平均能量輸出。

🧩

解析度 (Resolution)

如果解析度是 8-bit，代表我們可以將占空比分成 \(2^8 = 256\) 階（0–255）。解析度越高，控制越精細，但最高頻率也會隨之限制。

📡

頻率 (Frequency)

控制 LED 通常用 5000Hz (避開人眼閃爍與攝影機掃描頻率)；控制馬達則常用 1000Hz - 20000Hz。

講師手冊 (Instructor Guide)

1. 教學重點 (Teaching Points)

  • 強調 PWM 是一種「數位對類比的模擬」，電壓本質上仍是 0 與 3.3V。
  • 解釋工作週期 (Duty Cycle) 的計算公式：$D = \frac{T_{on}}{T_{total}}$。
  • 引導學生思考能量等效原理：平均電壓 $V_{avg} = V_{max} \times D$。

2. 常見問題 (Common Issues)

  • 誤以為電壓被降下來了：提醒學生若用三用電表測量，看到的是平均值，但示波器會顯示脈衝。
  • 頻率選取不當：頻率太低會導致肉眼可見的閃爍 (Flicker)；太高則會增加電磁干擾或驅動電路損耗。
  • 亮度不均：解釋 LED 亮度與電流呈現非線性關係，50% Duty Cycle 不代表一半的流明感。

3. 評分標準 (Grading Criteria)

  • 基礎分：能正確理解 Duty Cycle 並實作出呼吸燈效果。
  • 進階分：能解釋視覺暫留 (Persistence of Vision, POV) 如何使我們感知平滑的亮度變化。
