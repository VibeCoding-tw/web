解密 PWM 占空比魔法

理解如何透過極速開關切換，讓數位腳位模擬出連續的類比電壓感。

🌉 1. 數位轉類比的橋樑：什麼是 PWM？

開關的藝術：電腦腳位本質上是數位的，只能輸出 0V (OFF) 或 3.3V (ON)。

PWM (Pulse Width Modulation) 是透過極快速地切換開關，讓外部設備感受到一個「平均電壓」。

占空比 (Duty Cycle) 的奧祕：

在一個固定週期內，「開啟時間」佔「總週期時間」的百分比。

  • 25% Duty Cycle 1/4 時間開，3/4 時間關 → 感覺像是 3.3V 的 25%。
  • 50% Duty Cycle 一半開，一半關 → 感覺像是 3.3V 的一半。

⚡ 2. 能量等效原理：平均電壓

核心數學公式：

\[V_{avg} = V_{max} \times \text{Duty Cycle}\]

視覺暫留與慣性： 為什麼我們看不到燈在閃？

因為切換速度（頻率）通常高達數千赫茲 (Hz)，人眼與馬達轉子因為慣性來不及反應，只會感受到平滑的平均能量輸出。

🧩

解析度 (Resolution)

如果解析度是 8-bit，代表我們可以將占空比分成 \(2^8 = 256\) 階（0–255）。解析度越高，控制越精細，但最高頻率也會隨之限制。

📡

頻率 (Frequency)

控制 LED 通常用 5000Hz (避開人眼閃爍與攝影機掃描頻率)；控制馬達則常用 1000Hz - 20000Hz。

B. 參考範例代碼 (Reference Example Code)

main.cpp (基礎 LEDC 應用)

#include <Arduino.h>

const int ledPin = 2; // 選取具備 PWM 功能的 GPIO
const int freq = 5000;
const int ledChannel = 0;
const int resolution = 8; // 8-bit 解析度 (0-255)

void setup() {
  // 設定 LEDC 屬性
  ledcSetup(ledChannel, freq, resolution);
  // 將頻道連結至 GPIO
  ledcAttachPin(ledPin, ledChannel);
}

void loop() {
  // 呼吸燈效果
  for(int dutyCycle = 0; dutyCycle <= 255; dutyCycle++){   
    ledcWrite(ledChannel, dutyCycle);
    delay(7);
  }
}

C. 自動評分 (Autograding) 的設計

  • 核心語法檢索： 偵測程式中是否呼叫了 ledcWrite() 與 ledcSetup()。
  • 配置參數驗證： 驗證學生設定的解析度是否與其迴圈邊界 (如 0-255) 相互匹配。
