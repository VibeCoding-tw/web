CSS 圓形佈局與定位

完美的圓形容器

在網頁中製作圓形並不難，但要讓它完美置中且適應各種螢幕，就需要一點 CSS 技巧。

關鍵屬性：

  • border-radius: 50% 將方形變成圓形。
  • aspect-ratio: 1 / 1 確保寬高比永遠即是正方形。
  • transform: translate(-50%, -50%) 基於自身尺寸的中心點偏移。

Module 2: Core Principles

HTML5 Canvas 繪圖基礎

2D 渲染上下文

Canvas 就像一張空白的畫布，我們需要透過 JavaScript 取得 getContext('2d') 這支「畫筆」才能開始作畫。

JavaScript 
const canvas = document.getElementById('myCanvas');
const ctx = canvas.getContext('2d');

function draw() {
    // 1. 清除畫布 (必備！)
    ctx.clearRect(0, 0, canvas.width, canvas.height);

    // 2. 開始繪製路徑
    ctx.beginPath();
    ctx.arc(x, y, radius, 0, Math.PI * 2);
    ctx.fillStyle = 'blue';
    ctx.fill();
    
    // 3. 動畫循環
    requestAnimationFrame(draw);
}
  • 1
    座標系：

    原點 $(0,0)$ 位於畫布的左上角，X軸向右，Y軸向下。

  • 2
    路徑繪製：

    所有的繪圖都從 beginPath() 開始，這告訴瀏覽器我們要畫一個新的形狀。

Module 3: Practical Challenge

向量計算與實戰作業

預習：向量與角度 (Trigonometry)

要讓搖桿跟隨手指，我們需要計算兩點之間的距離與角度。

1. 角度計算 (Angle)

使用 Math.atan2(y, x) 計算滑鼠位置相對於圓心的角度。

\(\theta = \text{atan2}(dy, dx)\)

2. 距離計算 (Distance)

使用畢氏定理計算距離，確保搖桿頭不會超出底座範圍。

\(d = \sqrt{dx^2 + dy^2}\)
