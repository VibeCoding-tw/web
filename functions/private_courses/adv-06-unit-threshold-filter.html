數位影像特徵提取：
二值化與濾波

影像處理的第一步。學習如何透過數學手段，將複雜的現實世界轉化為機器易於理解的數位矩陣。

🌓 1. 影像二值化 (Thresholding) 的原理 (25 min)

核心概念：將灰階影像轉變為只有「黑」與「白」的過程。

臨界值 (Threshold)

設定一個標準，大於此值變白（背景），小於此值變黑（目標）。這能幫助電腦徹底排除光影雜訊。

  • ● 黑點與白點：電腦不再處理 256 種層次的灰，只處理 0 或 1。
  • ● 魯棒性：正確的臨界值能讓循跡線在不同光線下依然清晰。

🔍 學習任務

觀察不同臨界值對同一張「循跡線」圖片的影響——太高會讓線條斷裂，太低會讓背景雜訊干擾。

🧼 2. 影像濾波 (Filtering)：消除噪點的技術 (25 min)

● 均值濾波 (Mean Filter)

取周圍像素平均值。效果像「磨皮」，能讓影像變柔和，但會模糊邊緣。

● 中值濾波 (Median Filter)

取周圍像素的中位數。這是對抗「鹽椒雜訊（黑白點）」最強大的武器。

🧩 卷積核 (Kernel) 的概念

想像一個 $3 \times 3$ 的小方格如何在圖片上滑動並重算像素值。

🔬 3. 侵蝕、擴張與流程優化 (10 min)

● 形態學運算

學習使用「侵蝕 (Erosion)」去除極小雜點與「擴張 (Dilation)」填補斷裂線條。

作業指引 (Assignment Guide)

進階 6 - 影像過濾與二值化

1-1 實作「灰階化」濾鏡

角色扮演： 影像處理工程師。

執行步驟：

  1. 撰寫 JS 迴圈遍歷 ImageData 陣列。
  2. 使用公式 $Gray = 0.299R + 0.587G + 0.114B$。
  3. 將 R, G, B 三個通道都設為 $Gray$ 值，並將 $A$ 設為 255。

評分重點： 畫面是否正確轉化為黑白影像而不失真。

1-2 手動門檻值 (Thresholding) 調校

角色扮演： 視覺系統部署專員。

執行步驟：

  • 實作邏輯：若 $Gray < Threshold$，則設為 0；否則設為 255。
  • 建立 <input type="range"> 滑桿。
  • 觀察在不同光線環境下，使「黑線與背景」分離度最高的門檻數值。

評分重點： 能否從彩色畫面中，精確地「挖出」黑色的賽道線段。

1-3 反色過濾與自定義邏輯

角色扮演： 創意視覺設計師。

執行步驟：

  • 嘗試讓「深色變白，淺色變黑」，或者只保留特定通道的顏色。
  • 分析这种处理方式对于寻找白色物体（如白球）是否有幫助。

評分重點： 學生能否根據任務需求，調整代碼邏輯以解決特殊的偵測問題。
