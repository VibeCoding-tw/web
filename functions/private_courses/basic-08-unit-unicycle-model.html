基礎 8 - 運動學轉換

差速轉換數學：
Unicycle Model

將抽象的速度與轉向，解構成左右馬達的動力之源。

🚲 1. 什麼是 Unicycle Model？ (30 min)

核心概念：將車輛簡化為一個點，透過兩個參數來控制其運動模式。

v 線速度 (Linear Velocity)

代表車輛前進或後退的速度。在搖桿上，這通常對應到 Y 軸。

ω 角速度 (Angular Velocity)

代表車輛旋轉或轉向的速度。在搖桿上，這通常對應到 X 軸。

🎯 學習目標

理解為何搖桿的垂直位移控制速度，水平位移控制旋轉，這是建立直覺操控的基礎。

⚙️ 2. 差速驅動 (Differential Drive) 原理 (30 min)

物理結構：兩輪獨立驅動。要讓車轉向，左輪與右輪必須產生「速度差」。

基本混合公式

\[L = v - \omega\]
左輪轉速
\[R = v + \omega\]
右輪轉速

🧠 學習任務：腦力激盪

若 \(v=100\), \(\omega=50\)（代表前進中右轉），算出左、右輪各是多少？
（提示：左輪 50, 右輪 150）

🛑 3. 飽和度處理與比例縮放 (20 min)

⚠️ 問題：當數值溢出時？

若 \(v=255\) 且 \(\omega=100\)，算出的 \(R\) 會變成 \(355\)。但馬達最大 PWM 只有 \(255\)，直接裁切（Clipping）會導致轉向失靈。

預習概念：等比例縮放 (Normalization)

\[max\_val = \max(|L|, |R|, 255)\]
\[L_{final} = (L / max\_val) \times 255\]
\[R_{final} = (R / max\_val) \times 255\]

目標：即使在全速前進時，轉向指令依然能有效發揮作用，保持車輛的動態平衡。

講師手冊 (Instructor Guide)

教學重點 (Teaching Points)

  • 從幾何學切入：解釋 Linear Velocity ($v$) 影響前進，Angular Velocity ($\omega$) 影響轉彎的方向性。
  • 輪距 (Track Width) 的影響：解釋為什麼同樣的 $\omega$ 在不同寬度的車體上會產生不同的轉向手感。
  • 控制層級架構：強調 Unicycle Model 位於「策略層」與「馬達驅動層」之間。

學生常見問題 (Common Issues)

  • 計算結果超出 PWM 範圍：當 $v=1.0, \omega=1.0$ 時，其中一輪會變 2.0（需進行縮放處理）。
  • 左右輪方向相反：混控公式中的加減號與硬體接線不匹配。
  • 不理解原地旋轉 (Spin Turn) 的實現條件 ($v=0, \omega \neq 0$)。

評分標準 (Grading)

  • 優 (90-100): 指令與馬達響應完全一致，且能優雅地處理速度溢出的縮放問題閉。
  • 可 (70-89): 基礎運動正常，但對於極限值的處理不完全，導致在大轉彎時出現頓挫。
  • 待加強 (Below 70): 公式錯誤導致車子運動方向混亂（如：指令右轉卻左轉）。

在 GitHub Classroom 的實戰部署建議

A. 範本倉庫 (Template Repo) 結構

B. 自動評估 (Autograding) 的設計

  • 混控邏輯驗證： 給予 $v=0, \omega=1$，驗證馬達輸出是否為一正一負（原地旋轉）。
  • 歸一化檢核： 給予 $v=1, \omega=1$，驗證馬達輸出是否能縮放回 $\pm 100\%$ 以內。
