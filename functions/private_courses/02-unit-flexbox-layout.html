CSS Flexbox：遙控器排版的神器

告別 float 與 position 的噩夢，用彈性盒子打造完美的響應式遙控介面。

Module 1: Intuitive Lab

核心概念：容器與軸線

只要在父容器設定 display: flex;，它就變成了 Flex Container，裡面的子元素立刻變成乖巧的 Flex Items。

🔑 核心語法

↔️

主軸 (Main Axis)

決定排隊的方向。預設是水平向右 (`row`)。

↕️

交錯軸 (Cross Axis)

與主軸垂直的方向。決定元素如何"換行"或"堆疊"。

Module 2: Core Principles

對齊三劍客

1. justify-content

主軸上的分配。決定按鈕也是靠左、靠右，還是分散對齊。

space-between

2. align-items

交錯軸上的對齊。讓不同大小的按鈕在同一水平線居中。

center

3. flex-wrap

空間不夠時的策略。要硬擠在同一排，還是換行？

wrap

Module 3: Classroom Challenges

實戰挑戰：Flexbox 遙控器

作業 2-1

45 min

十字方向鍵佈局

目標：達成完美的幾何對稱。無論螢幕多寬，按鈕組永遠在正中央。

挑戰重點
  • 1. 外層容器使用 flex-direction: column 處理上下按鈕。
  • 2. 中間層嵌入 row 容器處理左右按鈕。
  • 3. 活用 align-items: center 確保垂直居中。
