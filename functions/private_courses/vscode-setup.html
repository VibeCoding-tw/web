🛠️ 直觀體驗：從零開始的基建

打地基的時間到了！讓我們用 15 分鐘完成最強編輯器的部署。

1
下載與安裝 (15 min)

前往 code.visualstudio.com，根據你的作業系統 (Windows/macOS/Linux) 下載。

Windows 用戶關鍵設定：

安裝時務必勾選 「加入 '開啟以程式碼編輯' 動作到 Windows 檔案總管內容功能表」 (Add "Open with Code")。

💡 這讓你能在資料夾直接按右鍵開啟專案，效率提升 10 倍！

2
繁體中文介面設定 (5 min)

  • 開啟 VS Code。
  • 點擊左側工具欄最下方的四方塊圖示 (Extensions) 或按 Ctrl+Shift+X。
  • 搜尋 Chinese，選擇 "Chinese (Traditional) Language Pack" 安裝。
  • 重啟 VS Code，介面即刻中文化。

🚀 實作挑戰：硬體大腦與雲端同步

✓

任務 2-1：安裝硬體大腦 PlatformIO (30 min)

這是開發 ESP32 無人車最重要的核心外掛。

  1. 擴充功能搜尋 "PlatformIO IDE" 並安裝。
  2. 注意： 安裝過程會自動下載 Python 與核心，請耐心等待。
  3. 驗證：安裝完成後，左側欄會出現一個「小外星人 👽」圖示。
✓

任務 2-2：Git 帳號同步 (20 min)

體驗「雲端同步、多機開發」的流暢感。

  1. 點擊左下角的人頭圖示 (Accounts)。
  2. 選擇 "Turn on Settings Sync..."。
  3. 登入你在家申請好的 GitHub 帳號。
