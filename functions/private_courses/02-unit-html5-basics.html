Module 2: Core Principles

HTML5 骨架與行動優先設計

無人車控制面板的構造 (HTML Structure)

HTML5 就像房子的鋼骨結構。為了讓機器人（無人車）能被理解，我們使用具有意義的標籤。

<> <header> 標題區

放置 Logo、連線狀態指示燈。告訴使用者「我在控制哪一台車」。

<> <main> 核心操作區

<video> ESP32-CAM 即時串流影像
<button> 前進、後退、轉彎控制
<input type="range"> 速度控制拉桿

<> <footer> 狀態資訊

顯示 IP 位址、版權宣告、除錯訊息。

行動優先 (Mobile First) 的關鍵

為什麼手機網頁字這麼小？

預設情況下，手機會假裝自己是 980px 寬的電腦螢幕，然後縮小顯示。這對操控遙控車是致命的。

<head>
<meta name="viewport"
content="width=device-width, initial-scale=1.0">
</head>
  • ✔ width=device-width: 強制網頁寬度等於手機螢幕寬度 (1:1)。
  • ✔ initial-scale=1.0: 初始縮放比例為 100%。

作業指引 (Assignment Guide)

作業 1-1：實機部署與跨裝置連線

  • 角色扮演： 部署工程師。
  • 執行方式： 學生 Clone 範本後，在 README.md 貼上手機成功連線電腦 Live Server 的截圖。
  • 評分重點： 確認學生能正確識別網段（WiFi）並完成跨端預覽。

作業 1-2：觸控友善性優化 (CSS)

  • 角色扮演： 介面工程師。
  • 執行方式： 在 style.css 中實作防止誤觸的屬性（如 touch-action）。
  • 評分重點： 是否使用了 user-select: none 與正確的按鈕間距。

作業 1-3：FPV 影像容器佈局

  • 角色扮演： 前端開發者。
  • 執行方式： 使用 object-fit 處理不同比例（16:9, 4:3）的影像容器。
  • 評分重點： 影像不變形且能夠滿板顯示。

GitHub Classroom 詳細操作指南

第一步：準備範本倉庫 (Template Repo)

  1. 在 GitHub 建立新 Repo（名為 html5-basics-template）。
  2. 上傳基礎結構。
  3. 在 Repo 設定中勾選 "Template repository"。

第二步：建立 Classroom 作業

  1. 進入 GitHub Classroom 建立新作業。
  2. 選擇「Individual Assignment」，並連結上述的範本倉庫。
  3. 重點設定：勾選 "Enable feedback pull requests"。

第三步：學生端操作 (Flipped Workflow)

  1. 學生點擊作業連結，獲得個人專屬 Repo。
  2. 本地開發後 Push 代碼。

第四步：教師回饋 (Review)

  1. 進入 Feedback PR 的 Files Changed。
  2. 針對學生的 CSS 優化細節給予專業建議。

在 GitHub Classroom 的實戰部署建議

建議範本倉庫設置如下：

A. 範本倉庫 (Template Repo) 結構

.
├── index.html       # 基礎 HTML 結構（含 meta viewport）
├── style.css        # 學生需在此填寫觸控優化 CSS
├── assets/          # 預置的 ESP32-CAM 測試影像
└── README.md        # 供學生貼上手機連線截圖

B. 參考範例代碼 (Reference Example Code)

index.html (基礎結構)

<!DOCTYPE html>
<html>
<head>
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <div class="video-container">
        <img src="assets/esp32-stream.jpg" id="live-feed">
    </div>
</body>
</html>

style.css (觸控優化)

.video-container {
    width: 100vw;
    height: 100vh;
    overflow: hidden;
    background: #000;
}
#live-feed {
    width: 100%;
    height: 100%;
    object-fit: contain; /* 確保不變形 */
}

  • 關鍵字檢查： 掃描 style.css 是否包含 width=device-width (雖然常在 HTML，但練習中可要求 CSS 註解或特定屬性)。
