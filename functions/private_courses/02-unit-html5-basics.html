本地伺服器 (Local Server)

模擬真實網站運作環境，讓你的電腦變成一台小型伺服器。

🔄

熱重載 (Hot Reload)

存檔的瞬間，瀏覽器自動更新畫面，無需手動 F5。

Module 2: Core Principles

HTML5 骨架與行動優先設計

無人車控制面板的構造 (HTML Structure)

HTML5 就像房子的鋼骨結構。為了讓機器人（無人車）能被理解，我們使用具有意義的標籤。

<> <header> 標題區

放置 Logo、連線狀態指示燈。告訴使用者「我在控制哪一台車」。

<> <main> 核心操作區

<video> ESP32-CAM 即時串流影像
<button> 前進、後退、轉彎控制
<input type="range"> 速度控制拉桿

<> <footer> 狀態資訊

顯示 IP 位址、版權宣告、除錯訊息。

行動優先 (Mobile First) 的關鍵

為什麼手機網頁字這麼小？

預設情況下，手機會假裝自己是 980px 寬的電腦螢幕，然後縮小顯示。這對操控遙控車是致命的。

<head>
<meta name="viewport"
content="width=device-width, initial-scale=1.0">
</head>
  • ✔ width=device-width: 強制網頁寬度等於手機螢幕寬度 (1:1)。
  • ✔ initial-scale=1.0: 初始縮放比例為 100%。
